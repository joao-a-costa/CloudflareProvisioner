# CloudflareProvisioner

CloudflareProvisioner is a .NET project designed to automate and manage Cloudflare resources and configurations. It supports both .NET Standard 2.0 and .NET 10, making it compatible with a wide range of .NET applications and environments.

## Features
- Automate Cloudflare DNS and security settings
- Manage Cloudflare zones and records programmatically
- Designed for extensibility and integration

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/joao-a-costa/CloudflareProvisioner.git
   ```
2. Open the solution in Visual Studio 2022 or later.
3. Build the solution to restore dependencies.

## Requirements
- .NET Standard 2.0 or .NET 10 compatible environment
- Visual Studio 2022 or later recommended
